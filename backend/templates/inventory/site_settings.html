Activar HTTPS

Cuando guardes el certificado, ejecuta estos pasos como root en el servidor de monitoreo.

  1. Exporta el certificado cifrado hacia la carpeta segura del sistema.
  2. Instala la configuracion Nginx incluida con la plataforma.
  3. Habilita el sitio y recarga Nginx.
cd /opt/monitoring-platform/backend
set -a; source /etc/monitoring-platform.env; set +a
.venv/bin/python manage.py export_tls_certificate
sudo cp /opt/monitoring-platform/deploy/nginx-https.conf /etc/nginx/sites-available/monitoring-platform
sudo ln -sfn /etc/nginx/sites-available/monitoring-platform /etc/nginx/sites-enabled/monitoring-platform
sudo nginx -t && sudo systemctl reload nginx

Antes de usarlo, reemplaza MONITOR.DOMINIO.O.IP en el archivo Nginx por el dominio o IP informado arriba. El nombre debe coincidir con el certificado.
