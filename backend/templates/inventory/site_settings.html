Activar HTTPS en Nginx

Si el certificado ya fue instalado manualmente en Nginx, el portal puede funcionar con HTTPS aunque esta seccion indique que no hay certificado almacenado.

  1. Exporta el certificado guardado en la plataforma hacia la carpeta segura del sistema.
  2. Instala la configuracion Nginx incluida con la plataforma.
  3. Valida la configuracion y recarga Nginx.
cd /opt/monitoring-platform/backend
set -a; source /etc/monitoring-platform.env; set +a
.venv/bin/python manage.py export_tls_certificate
sudo cp /opt/monitoring-platform/deploy/nginx-https.conf /etc/nginx/sites-available/monitoring-platform
sudo ln -sfn /etc/nginx/sites-available/monitoring-platform /etc/nginx/sites-enabled/monitoring-platform
sudo nginx -t && sudo systemctl reload nginx

Para que la aplicacion detecte HTTPS correctamente detras de Nginx, la configuracion debe enviar la cabecera X-Forwarded-Proto https.
